# 支持向量机分类示例
GitHub：https://github.com/lanling-47/Machine-Learning.git
## 项目简介

本项目演示了如何使用支持向量机（SVM）进行分类任务，并绘制决策边界。使用了 NumPy、Matplotlib 和 scikit-learn 库。
使用Jupyter编程，代码描述放在markdown中。

## 依赖

在运行项目之前，请确保安装以下 Python 包：

- numpy
- matplotlib
- scikit-learn

您可以使用以下命令安装所需的包：

```bash
pip install numpy matplotlib scikit-learn
